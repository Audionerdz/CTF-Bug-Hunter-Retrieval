### Cheatsheet Hotkeys Linux 

| Acción                                                   | Atajo        |
| -------------------------------------------------------- | ------------ |
| Mover a inicio de la línea                               | **Ctrl + A** |
| Mover al final de la línea                               | **Ctrl + E** |
| Mover una palabra a la izquierda                         | **Alt + B**  |
| Mover una palabra a la derecha                           | **Alt + F**  |
|                                                          |              |
| Acción                                                   | Atajo        |
| Borrar desde el cursor hacia atrás (como Ctrl+Backspace) | **Ctrl + W** |
| Borrar desde el cursor al final                          | **Ctrl + K** |
| Borrar desde el cursor al inicio                         | **Ctrl + U** |
| Borrar palabra hacia delante                             | **Alt + D**  |
|                                                          |              |
