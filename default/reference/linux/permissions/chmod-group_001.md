### **Permisos archivos al grupo (g)**

```cs
Solo lectura:  
`chmod g+r archivo`

Solo escritura:  
`chmod g+w archivo`

Solo ejecución:  
`chmod g+x archivo`

Lectura + ejecución:  
`chmod g+rx archivo`

Lectura + escritura:  
`chmod g+rw archivo`

Escritura + ejecución:  
`chmod g+wx archivo`

Lectura + escritura + ejecución:  
`chmod g+rwx archivo`

Quitar permisos:  
`chmod g-rwx archivo`
```
