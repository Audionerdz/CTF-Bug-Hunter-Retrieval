### **Permisos archivos al dueño (u)**

```cs
Dar solo lectura:  
`chmod u+r archivo`

Dar solo escritura:  
`chmod u+w archivo`

Dar solo ejecución:  
`chmod u+x archivo`

Dar lectura + ejecución:  
`chmod u+rx archivo`

Dar lectura + escritura:  
`chmod u+rw archivo`

Dar escritura + ejecución:  
`chmod u+wx archivo`

Dar lectura + escritura + ejecución:  
`chmod u+rwx archivo`

Quitar permisos:  
`chmod u-rwx archivo`
```
