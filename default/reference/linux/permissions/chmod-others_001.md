### **Permisos archivos a otros**

```cs
Solo lectura:  
`chmod o+r archivo`

Solo escritura:  
`chmod o+w archivo`

Solo ejecución:  
`chmod o+x archivo`

Lectura + ejecución:  
`chmod o+rx archivo`

Lectura + escritura:  
`chmod o+rw archivo`

Escritura + ejecución:  
`chmod o+wx archivo`

Lectura + escritura + ejecución:  
`chmod o+rwx archivo`

Quitar permisos:  
`chmod o-rwx archivo`
```
