# Query: Buscar Comandos Git

## Git Status
```bash
query "git status"
```

## Git Add y Commit
```bash
query "git add commit"
```

## Git Push Pull
```bash
query "git push pull origin"
```

## Git Branches
```bash
query "git branch checkout"
```

## Git Undo
```bash
query "git restore reset revert"
```
