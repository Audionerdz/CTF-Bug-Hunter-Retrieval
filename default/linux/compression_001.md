### Compresión y Manejo de Archivos en Linux

Comandos básicos para comprimir y descomprimir archivos en Linux, útiles para transferencia de datos o archivado.

```bash
# Comprimir carpeta en tar.gz
tar -czf archivo.tar.gz Lock/

# Descomprimir archivo tar.gz
tar -xvzf archivo.tar.gz

# Descomprimir zip
unzip extended-note.zip

# Comprimir carpeta en zip
zip -r skills.zip skills/
````
