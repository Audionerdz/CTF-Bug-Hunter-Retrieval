## Quick Reference: Main Commands

```bash
# Check status
git status

# See changes
git diff

# Stage everything
git add .

# Commit
git commit -m "message"

# Push
git push origin main

# Pull
git pull origin main

# View history
git log --oneline -10

# Undo changes
git restore .

# Revert commit
git revert commit-hash
```
