# Guía de uso para script de automatización de escaneo (autoscan.sh)

`autoscan.sh` es una utilidad personalizada diseñada para agilizar la fase de reconocimiento inicial, automatizando el lanzamiento de herramientas de escaneo (como nmap) contra un objetivo específico.


### Ejecución básica:

```bash
autoscan.sh <IP_DEL_OBJETIVO>

```

## 2. Ejemplo Práctico

Para iniciar el escaneo contra una máquina de HTB o un entorno local:

```bash
autoscan.sh 10.10.11.23

```
