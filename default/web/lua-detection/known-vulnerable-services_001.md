### Servicios Conocidos que Usan Lua

- **OpenResty** - Nginx con Lua integrado para APIs y lógica de servidor
- **Wing FTP Server** - Servidor FTP que procesa sesiones con Lua
- **Tarantool** - Base de datos con scripting en Lua
- **Roblox** - Plataforma de juegos que usa Lua para scripts
- **FiveM** - Framework para servidores de GTA V que usa Lua

Cualquiera de estos servicios es un objetivo potencial si está expuesto en un puerto web o FTP.
